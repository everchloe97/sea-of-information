1. Binary Tree - 자식 node를 **최대 2개** 가질 수 있는 Tree

2. B-Tree (Balanced Tree) - 자식 node를 2개 이상 가질 수 있는 Tree <br/>

3. B+Tree - 같은 레벨의 형제 node는 연결 리스트 형태로 이어져 있는 Tree
   *index node
   *data node
